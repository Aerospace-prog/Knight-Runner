# Knight Runner

Welcome to the Knight Runner project!

## Overview
Knight Runner is an exciting adventure game where players take on the role of a knight navigating through various challenges and obstacles. The goal is to collect treasures while avoiding enemies and traps.

## Features
- Engaging gameplay with multiple levels
- Rich graphics and sound effects
- Leaderboard to track high scores

## Installation
To get started with the Knight Runner, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/Aerospace-prog/Knight-Runner.git
cd Knight-Runner
```

## Usage
Run the game using:

```bash
python main.py
```
